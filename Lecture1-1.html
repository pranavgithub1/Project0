CSS

If you had an ordered html list and wanted to style items in the list (descendants) then you would do:


ol li { }

Which means to select all list elements within that are descendants of ordered lists. The space operator selects any descendants of an element. This will give all descendants of the ordered list (including sub unordered lists). If you only want direct descendants use “>” instead of “ “.

input[type=”text”] { background-color: red; }

means all input fields of type text should be red

Psuedo-classes and psuedo-elements button { some styling background-color: red; } button:hover{ background-color: green; }

“:hover” is a psuedo-class. This allows the button to change color when you hover it.

HTML: <a href="#"> a link </a> css: a::before { content: "\21d2 Click here: "; font-weight: bold; }

::before is a psuedo element that allows you to insert next before an element. The above code produces the following result:

a link

Another cool psuedo-element is ::selection

p::selection { color: red; background-color: yellow; }

To see what the code above does, try selecting this text.

Sass (.scss)

A language built on top of css so we can programatically generate style sheets.

For example if we wanted to use the same color to style many elements and be able to change it easily we could define a variable:

$color: red;

Now everywhere we put $color it will be interpeted as red.

We can also use styling inheritance with sass:

%message { font-family: sans-serif; font-size: 18px; font-weight: bold; border: 1px solid black; padding: 20px; margin: 20px; } .success { @extend %message; background-color: green; } .failure { @extend %message; background-color: red; } .warning { @extend %message; background-color: yellow; } HTML: <div class="success">Success</div> <div class="failure">Failure</div> <div class="warning">Warning</div>
