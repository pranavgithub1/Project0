CSS

CSS is what we use to style our pages, and make them look nice.
Take a look at this example code:
<h6 style="color:blue"> Heading 1 </h6> This will give the h6 a blue color:

Heading 1

The color is a css property, however thing get clutered if we style within the HTML.

Therefore, we link a seperate .css file called a stylesheet:
<link rel="stylesheet" href="style.css">

CSS has many ways of selecting what HTML elements you want to style.

You can refer to them by name: div{ property: value; }

You can name them in the form of classes and ids: .oranges { /* refers to elements with class "title" */ } #apples { /* regers to elements with id "password" */ } You can give html elements ids and classes like attributes: <div class="oranges"></div> <p id="apples"></p> Classes and Ids exist for easy selection.
